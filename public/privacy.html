2. 我们收集什么信息

2.1 您主动提供的信息

  • 账号信息：邮箱、公司名称、联系电话（如填）、访问令牌
  • 业务信息：产品类型、产品优势、目标客户描述（ICP）
  • 搜索条件：关键词、目标城市、目标国家
  • 推送配置：Instantly Campaign ID

2.2 我们自动收集的信息

  • 使用日志：登录时间、操作记录、搜索历史
  • 技术信息：IP 地址、浏览器类型、设备信息

2.3 我们 不收集：

  • 您的 API key（智拓客使用内置 API、不需要您提供）
  • 支付信息（暂未启用付费功能）
  • 您的客户/线索数据中的敏感个人信息（如身份证号、银行卡号）

3. 我们如何使用您的信息

  • 提供拓客、AI 生成开发信、推送 Instantly 等核心服务
  • 账户管理与身份验证
  • 服务质量改进与故障排查
  • 发送服务相关通知（不发送营销邮件）
  • 遵守法律法规要求

4. 信息存储与跨境传输

重要：智拓客的服务器位于亚洲数据中心（Hostinger），您的数据将存储在境外服务器上。

4.1 数据存储位置

  • 主数据库：Supabase（云数据库）
  • 应用服务器：Hostinger（亚洲数据中心）

4.2 第三方服务提供商

为提供服务，我们将必要数据传输给以下第三方：

  • Anthropic（美国）：AI 评分与邮件生成 · 传输：客户公司信息、网站内容
  • Apify（捷克）：Google Maps 数据抓取 · 传输：搜索关键词
  • Firecrawl（美国）：网站内容抓取 · 传输：目标网站 URL
  • Instantly（美国）：邮件推送 · 传输：邮件正文、收件人邮箱

这些服务商均与我们签署了保密协议，您的信息仅用于为您提供服务、不会被用于其他目的。

  • 在每封邮件中提供有效的退订机制
  • 不发送欺骗性、误导性内容
