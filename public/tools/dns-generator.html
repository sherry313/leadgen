2. 输入你的域名
3. DMARC 策略
none — 先监控，不拦截
推荐新手先用这个
quarantine — 丢垃圾箱
可疑邮件进垃圾箱
reject — 直接拒收
最严格，冒充邮件门都别进
4. 接收报告的邮箱
SPF 发件人验证
Type TXT
Host @
Value
—
DKIM 数字签名
Type TXT
Host —
Value 请从你的邮件服务商后台复制 DKIM 密钥
⚠ DKIM 密钥每个域名不同，无法自动生成。请登录你的邮件服务商后台复制公钥（public key），完整长度通常 200+ 字符。
DMARC 综合策略 + 报告
Type TXT
Host _dmarc
Value
—
📌 设置步骤
  1. 登录你的域名服务商（Namecheap、阿里云、腾讯云）
  2. 找到 DNS 设置 / Advanced DNS
  3. 添加上面三条 TXT 记录
  4. 等待 15-30 分钟生效
  5. 去 mail-tester.com 测试
